Manual de Operador

1. Cómo buscar documentos

El buscador es inteligente. No necesitas escribir acentos ni mayúsculas.

Ejemplo 1: Si escribes "rumania", el sistema encontrará "Rumanía".
Ejemplo 2: Escribe "PNC" para Permisos de Conducción o "PSP" para Pasaporte.
2. El Visor de Imágenes

Al pulsar el icono del ojo, se abre el documento en pantalla completa.

  • Zoom: Usa dos dedos o la rueda del ratón.
  • Navegación: Usa las flechas para ver el siguiente documento del mismo país.
3. Estadísticas

Arriba verás un contador de archivos Nuevos (últimas 24h) y el Total.
